A memory engine for AI agents on Qwen Cloud. One turn: sweep decay, recall under budget, answer, write through adjudication.

↓
Memory engine (TypeScript core, 67 unit tests) Recall: similarity + retention + importance + frequency, greedy-packed under a hard token budget (never exceeded). Forgetting: Ebbinghaus retention decay, type half-lives, decay sweep each turn. Decayed, not deleted. Truth maintenance: similarity-gated contradiction adjudication; superseded facts keep an audit pointer. Consolidation: session transcript distilled to typed durable memories.
